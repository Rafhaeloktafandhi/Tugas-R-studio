# Tugas-Tugas Model LInier

## deskripsi
- Tugas 1: mencari variansi x dan y, kovariansi xy, nilai beta 1 dan beta2.  
- Tugas 2: mencari nilai t, P-value, dan selang kepercayaan  
- Tugas 3: mencari nilai R-squared, SSE, SSR, SST, df1, df2, dan F.  
- Tugas 4: mencari estimasi beta0, beta1, dan beta2 dari variabel prediktor kontinu stress level dan jam belajar perminggu terhadap total nilai serta model prediksinya.  
- Tugas 5: mencari inter pretasi koefisien intercept dan slope .  
- Tugas 6: mencari estimasi model dari variabel kontinu (stress level) dan kategorik (gender) terhadap total nilai.  
- Tugas 7: membandingkan full model dan reduced model.
- Tugas 8: melakukan Uji Normalitas dengan Shapiro Wilk, Uji Homoskedastisitas dengan Breusch Pagan, Uji Independent dengan Durbin Watson, dan Linearitas
- Tugas 9: mencari nilai RMSE dari model full dan model reduced
